# Usage

## Load a profile

Use **Charger un profil JSON** and select a profile file. Profiles can contain text fields, typography, logo data, and social links.

## Thunderbird setup

1. Open Thunderbird account settings.
2. Select the email account.
3. Enable “Attach the signature from a file”.
4. Select `signature-base64.html`.

## Social buttons

Enable only the networks you want to show, then enter the full profile URL. Generated buttons are regular clickable links.

## Save a profile

Use **Sauver ce profil en JSON**, enter a profile name, and the browser downloads `<profile-name>.json`.
