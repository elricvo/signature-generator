Aide

    
Ce générateur crée une signature HTML compatible Thunderbird. La version recommandée est signature-base64.html, car elle embarque le logo directement dans le fichier.

    
    
      
  1. Optionnel : charger un profil JSON pour préremplir le générateur, ou sauver la configuration courante avec Sauver ce profil en JSON.
    2. 
      
  2. Choisir ou remplacer le logo si nécessaire.
    3. 
      
  3. Remplir les champs de texte utiles.
    4. 
      
  4. Ajuster la police, la taille, la couleur, la graisse et l’italique champ par champ.
    5. 
      
  5. Optionnel : sélectionner les réseaux sociaux à afficher et renseigner les liens de profils.
    6. 
      
  6. Cliquer sur Prévisualiser.
    7. 
      
  7. Télécharger signature-base64.html.
    8. 
      
  8. Dans Thunderbird : Paramètres des comptes → Apposer la signature à partir d’un fichier → sélectionner signature-base64.html.
    9. 
    

    
La variante signature.html + logo séparé fonctionne aussi, mais le logo doit rester dans le même dossier que le fichier HTML.

    
Conseil : pour éviter les déformations, le logo est généré avec une largeur fixe et une hauteur automatique.
